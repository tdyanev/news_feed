# News web portal

This project lets you create your own news portal.

## Requirements

* PHP 7.0+
* `composer` command (See [Composer Installation](https://getcomposer.org/doc/00-intro.md#installation-linux-unix-osx))
* Git

## How to Use

### Install 
```
$ composer install
```

### Run PHP built-in server

```
$ bin/server.sh
```
